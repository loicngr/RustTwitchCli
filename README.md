# Available commands

### Env vars (.env)
    TWITCH_CLIENT_ID=
    TWITCH_CLIENT_SECRET=
    TWITCH_CLIENT_TOKEN=

### Generate Token
- > cargo run token

### Get User by id
- > cargon run info-user=53380605

### Get user Stream by id
- > cargo run isonlive-user=147337430

### Get an user id by user login
- > cargo run uid=username
